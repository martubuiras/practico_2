# ¿Cómo instalar las dependencias y ejecutar el programa del ejercicio 10?

Los requisitos para lograrlo son:

* Python 3
* Jupyter Notebook


Instalación de las dependencias:

En git bash realizar los siguientes pasos

1. git clone <URL_DEL_REPOSITORIO>

2. cd mi_proyecto
   
3. python -m venv venv
   
4. source venv/Scripts/activate

5. pip install notebook


Ejecución del programa:

En git bash escribir

- jupyter notebook

Y luego abrir

- notebooks/Ejercicio10.ipynb

Y ejecutar todas las celdas.
